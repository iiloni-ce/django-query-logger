# Tech Stack

## Runtime & Package Management
- **Bun**: Fast JavaScript/TypeScript runtime and package manager used for both backend and frontend development.

## Backend
- **Fastify**: High-performance web framework for Node.js/Bun.
- **Fastify WebSocket**: For realtime communication between server and clients.

## Frontend
- **Angular (v18)**: Framework for building the single-page application.
- **Angular Material**: Component library for UI elements.
- **SCSS**: For styling components.

## Languages
- **TypeScript**: Used exclusively across the codebase for type safety and maintainability.

## Key Libraries
- **pgsql-ast-parser**: For parsing and analyzing PostgreSQL queries.
- **pickleparser**: To handle data ingested from Django's socket logging handler.
- **luxon**: For date and time manipulation in the UI.
- **ngx-highlightjs**: For syntax highlighting of SQL queries.
